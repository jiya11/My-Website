Crazy Taxi Project

Overview

This project is a hardware implementation of the classic Crazy Taxi arcade-style game, reminiscent of the popular version on CoolMathGames, built entirely from scratch in C and deployed on an FPGA development board. The game features custom graphics rendering through VGA display output, real-time input processing via PS/2 keyboard, and dynamic scoring displayed on a seven-segment display.

Software Architecture

  • Double-buffered Graphics Engine: Implements vsync-based buffer swapping to eliminate screen tearing
  • Custom Pixel Plotting System: Directly writes to memory-mapped VGA buffers
  • Hardware-Accelerated Input Processing: Real-time keyboard input handling through memory-mapped I/O
  • Physics System: Implements collision detection and jumping mechanics

Gameplay Elements

  • Lane Switching: Players can switch between left and right lanes using arrow keys
  • Jump Mechanics: Press spacebar when approaching obstacles to jump over them
  • Scoring System: Points are earned for each obstacle successfully avoided
  • Progressive Difficulty: The game gradually increases in difficulty as the player advances
